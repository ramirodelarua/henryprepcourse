* `for`
* `&&`, `||`, `!`


'For': Este recurso de javascript se utiliza cuando necesitamos repetir un proceso las veces que consideremos necesarias, hasta que se cumpla lo que queremos lograr.

'&&': Cuando le pedimos al intérprete de javascript que considere más de una condición para evaluar, ponemos '&&' ("y", o "and" en inglés) entre las condiciones.
'||': Cuando le pedimos al intérprete de javascript que considere una condición u otra a la hora de evaluar (a modo excluyente), ponemos '||' ("o", o bien "or" en inglés) entre las condiciones.
'!' : Se antepone un '!' a una condición o signo cuando queremos negar o indicar lo opuesto a ese contenido.
